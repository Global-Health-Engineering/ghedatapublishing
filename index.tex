\chapter{Creating a repository}\label{creating-a-repository}

First, you have to create a repository within the GHE organization. You
can do this here:

Choose a new for your repository, make it public if you're data is not
confidential/sensitive and click on ``Create repository''. For now, now
README, .gitignore file or a license is need. You'll add these at a
later stage.

Once the repo is initialized, go back to RStudio and check if the
\texttt{devtools} and \texttt{usethis} packages are installed. If not,
please install/updated them now as some of their functions are needed in
the next steps.
